## Module Prim.Boolean

The Prim.Boolean module is embedded in the PureScript compiler. Unlike `Prim`, it is not imported implicitly. It contains a type level `Boolean` data structure.
#### `True`

``` purescript
data True :: Boolean
```

The 'True' boolean type.

#### `False`

``` purescript
data False :: Boolean
```

The 'False' boolean type.
